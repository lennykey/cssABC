What does it do:
Reads the actual view and writes result to a new tab

Install:
Please clone this repository in the Packages folder of sublime.

Fon example in Ubuntu:
cd ~/.config/sublime-text-2/Packages/

git clone https://github.com/lennykey/cssABC.git

Execution of this plugin:
CTRL+SHIFT+A (release this combo) and then press B (release) and then C (mnemonic for cssABC)
